How to run:
unzip the A3_36.tar file
make sure you have make installed
go into terminal into the folder A3_36
give the command:
$make
this will create a lexical analysis file output.txt
We have followed the Flex I/O (interactive) format in the slides.
Note:
1) We have taken 0 as an Interger Constant.
2) For string literal double quatation(") at the ends are excluded.
3) For character constant (') at the ends are excluded.
